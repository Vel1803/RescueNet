# 🚨 RescueNet

## 🎙️ Voice Emergency Response System

RescueNet is a voice-based emergency response system designed to help users act quickly during critical situations such as accidents, fires, or crimes. Instead of typing or calling, users can simply speak, and the system responds instantly.

---

## 💡 Problem

In emergency situations, people often panic and are unable to take quick actions. Many users struggle to call emergency services or use mobile apps properly during high-stress moments. Existing systems mainly focus on reporting incidents but do not provide real-time guidance or connect nearby people who can help.

---

## 🚀 Solution

RescueNet solves this problem by using voice as the primary interface. Users can report emergencies simply by speaking. The system:

- Detects the type of emergency
- Captures and displays user location
- Sends alerts to nearby people (simulated)
- Provides real-time voice guidance using AI

This makes the response faster, easier, and hands-free.

---

## 🗣️ How It Works

1. User clicks the microphone button  
2. Speaks the emergency (e.g., "Accident", "Fire")  
3. System processes the input  
4. Displays alert and location  
5. Provides voice instructions using AI  

---

## 🔧 Features

- 🎤 Voice Input (Speech Recognition)
- 🔊 AI Voice Output (Murf Text-to-Speech)
- 📍 Location Detection
- 🚨 Emergency Alert System (UI Simulation)
- ⚡ Real-time Response

---

## 🛠️ Tech Stack

- HTML, CSS, JavaScript  
- Web Speech API (Voice Input)  
- Murf AI API (Text-to-Speech)  
- Geolocation API  

---

## 💻 Requirements

- Device with microphone  
- Modern web browser (Chrome recommended)  
- Internet connection  

---
